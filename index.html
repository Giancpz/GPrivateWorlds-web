GPrivateWorlds es un plugin para que los jugadores puedan crear sus mundos privados, invitar a sus amigos, y visitar otros mundos.

Easy to use

Creado para ser fácil de usar, para el servidor más pequeño, hasta el servidor más grande.

Performance

Siempre teniendo como prioridad tener el mejor rendimiento y no tener funciones innecesarias.

Escalable (experimental)

En desarrollo para ser escalable entre muchos servidores con una configuración sencilla.
